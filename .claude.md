# Djangobee Prints Website - Claude Context

## Project Overview
This is the Djangobee Prints website, showcasing meaningful technology integration in elementary education. The site features educational thoughts and practical technology guides for educators.

## Project Purpose
- Share educational philosophy and classroom reflections
- Provide practical technology integration guides for elementary teachers
- Build a resource library for meaningful tech use in education

## Content Types
1. **Technology Guides** (`/tech-guides`) - Practical how-to tutorials, tool reviews, classroom setup guides
2. **Educational Thoughts** (`/thoughts`) - Blog-style posts about teaching philosophy, reflections, insights

## Key Workflow Documents
- **`plan.md`** - Overall project structure and technical considerations
- **`new_post_workflow.md`** - Step-by-step workflow for creating new posts (READ THIS when creating content)

## Content Creation Workflow
When asked to create a new post:
1. **ALWAYS reference `new_post_workflow.md` first**
2. Follow the structured process for handling text, images, and metadata
3. Maintain consistent formatting and organization
4. Place files in appropriate directories based on content type
